Input Container

You can easily put the the <md-chip-list> inside of an <md-input-container>.

{{person.name}} cancel

The example above has overridden the [separatorKeys] input to allow for ENTER, COMMA and SEMI COLON keys.

Autocomplete Example

You can also utilize the <md-chip-list> alongside the [md-autocomplete] component.

currentFood: {{currentFood}}

{{favoriteFood}} cancel {{ food }}

Note: If you want to allow only choices from the autocomplete, simply remove the mdChipInput directive and the (chipAdded) event.
